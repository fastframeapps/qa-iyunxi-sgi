# Pruebas QA - SGI Grupo Iyunxi

Colección de pruebas automatizadas desarrolladas como parte del plan de mejoramiento 
del proceso de aseguramiento de calidad (QA) en el Sistema de Gestión Inmobiliaria (SGI) 
de Grupo Iyunxi S.A.S.

## Contenido

- `SGI_QA_Tests.json` — Colección de Postman con 4 casos de prueba automatizados
- Casos cubiertos: consulta de convenios, registro de convenios, validación de errores e inmuebles

## Cómo ejecutar

1. Importar `SGI_QA_Tests.json` en Postman
2. Ejecutar con Collection Runner
3. Revisar resultados PASS/FAIL en el reporte generado
